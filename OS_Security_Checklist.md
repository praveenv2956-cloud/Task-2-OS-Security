# OS Security Checklist (Ubuntu)

## User Management
- Checked users using /etc/passwd
- System accounts use nologin

## File Permissions
- /etc/passwd permissions: -rw-r--r--
- Owned by root

## Processes & Services
- Verified running processes
- Reviewed active services

## Firewall
- UFW enabled
- Status: Active

## Security Principles
- Least privilege followed
